# Repository Structure

```text
StorageAI/
│
├── apps/              # Desktop application (React + Tauri)
├── assets/            # Icons, logos, fonts, images
├── docs/              # User documentation
├── Engineering/       # Active development workspace
├── Planning/          # Sprint planning and project planning
├── ProjectBrain/      # Permanent product knowledge
├── releases/          # Release builds and release notes
├── scripts/           # Development scripts
├── tests/             # Unit, integration and benchmark tests
├── tools/             # Internal developer tools
│
├── README.md
├── LICENSE
└── .gitignore
```

## Documentation

- **ProjectBrain/** contains permanent product knowledge.
- **Engineering/** contains active development work.
- **Planning/** contains sprint planning and project management.
- **docs/** contains user-facing documentation.
